  <h3>WordPress 暴力膜插件</h3>
  <p>这不是一个普通的插件，它象征着一代人的膜蛤精神，以及大家对长者的热情，它会在前台或者后台每个页面显示一句来自“怒斥香港记者”的长者经典语录，不但能够让访客变得Excited，还能让你的博客变得更快。</p>
</center>
<p>使用方法：丢进 plugins 文件夹即可。</p>
